Projects and Stuff

Blog

I post everything front end on my blog, from tips for responsive design to full guides on creating a workflow that suits your development needs. My posts are categorised and tagged with keywords to make it easy to find exactly what you're looking for.

From a development perspective, it's built on the Nunjucks templating engine and is precompiled with Gulp.js. The majority of pages are built automatically from JSON which is generated with a TinyMCE editor app.

NookApp

A hub of interactive apps to use alongside the Animal Crossing: New Horizons game on Nintendo Switch, including a Critterpedia, fossil encyclopedia and a daily checklist.

The app is created using Vue.js as the framework with Vuex as the state management pattern. The app is compiled and bundled using Webpack with the vue-cli configuration.

Find My Doughnuts

Using the Google Maps API, Find My Doughnuts will try and find doughnut stores near you using the device's geolocation.

This was the first app I made, and serves a good reminder of my progress in web development. Plus, I don't have the heart to change any of it because it's so funny as a concept :)
